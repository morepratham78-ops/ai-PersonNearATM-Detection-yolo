# AI Person Near ATM Detection using YOLO

Real-time AI detection system built using YOLO and Roboflow to detect people near ATM environments.

## Features
- Real-time person detection
- Bounding box visualization
- Confidence score prediction
- Custom dataset training

## Tech Stack
- Python
- YOLO
- Roboflow
- OpenCV
- Google Colab

## Workflow
1. Dataset collection
2. Data annotation using Roboflow
3. Model training using YOLO
4. Real-time detection testing

## Results
The model successfully detects people near ATM environments in real-time video streams.
